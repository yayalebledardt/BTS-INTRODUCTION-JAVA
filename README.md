# BTS - Introduction à Java

Ce dépôt contient les exercices et les projets réalisés dans le cadre du cours d'introduction à la programmation Java pour le BTS. L'objectif est d'apprendre les bases du langage Java et les concepts de la programmation orientée objet (POO).

## 🚀 Objectifs du cours

* Comprendre la syntaxe de base de Java.
* Manipuler les variables et les types de données.
* Utiliser les structures de contrôle (boucles et conditions).
* Apprendre les bases de la Programmation Orientée Objet (Classes, Objets).

## 📂 Structure du projet

* `src/` : Contient les fichiers sources `.java`.
* Chaque dossier ou fichier correspond à un chapitre ou un exercice spécifique du programme.

## 🛠️ Prérequis

Pour exécuter ces programmes, vous avez besoin de :
* **JDK** (Java Development Kit) 17 ou plus récent.
* Un IDE comme **IntelliJ IDEA**, **Eclipse** ou **VS Code**.

## 💻 Installation et Utilisation

1.  **Cloner le dépôt :**
    ```bash
    git clone https://github.com/yayalebledardt/BTS-INTRODUCTION-JAVA.git
    ```
2.  **Ouvrir le projet :**
    Ouvrez le dossier avec votre éditeur de texte ou IDE préféré.
3.  **Compiler et exécuter :**
    Vous pouvez compiler un fichier via le terminal :
    ```bash
    javac NomDuFichier.java
    java NomDuFichier
    ```
